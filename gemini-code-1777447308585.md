# Gandoura Stock Manager - Antigravity Skill

## System Context
Private inventory management app for traditional Moroccan clothing (Gandouras).
Role: Admin/Owner only.

## Technical Requirements
- UI: High-tech Minimalist (Dark theme, cyan accents, glassmorphism).
- Logic: Automatic sizing based on physical garment dimensions.
- Database: Products with {image, color, stock_initial, stock_sold, length, chest, shoulder, person_height, size}.

## Automated Logic Engine (The Moroccan Standard)
1. IF `garment_length` is [142-145] THEN `person_height` = "1m75 - 1m80" AND `size` = "L".
2. IF `garment_length` is [150-155] THEN `person_height` = "1m85 - 1m90" AND `size` = "XL".
3. IF `garment_length` >= 160 THEN `person_height` = "1m95+" AND `size` = "XXL".

## UI/UX Rules
- Dashboard must use "Grid of Cards".
- Each card has a visual indicator of "Remaining Stock" (Green/Red).
- Filter system by "Person Height" is a priority for customer service.
